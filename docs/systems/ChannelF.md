- Open supported Channel F ROM files from the File menu.
- Use save states and rewind to iterate quickly on difficult gameplay sections.
- Record and replay movies for deterministic verification.
- Open TAS editor for frame-level input editing and route branching.
- Use debugger windows for disassembly, memory inspection, trace, and event flow checks.
