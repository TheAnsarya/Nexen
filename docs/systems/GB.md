# Game Boy / Game Boy Color

## Overview

Nexen supports Game Boy and Game Boy Color emulation with movie recording, TAS editor tooling, save states, rewind, and debugger workflows.

## Hardware Focus

- CPU: LR35902 instruction behavior
- Video: handheld display timing model
- Audio: channel-based handheld audio behavior
- Cartridge: banked memory and cart metadata handling

## Using GB/GBC Features in Nexen

- Open ROM: select a .gb or .gbc file.
- Save states and designated slots: F1 and F4 workflows.
- Rewind and frame stepping: Backspace and backtick.
- TAS editing: use piano roll plus branch management for route testing.

## Accuracy and Compatibility

- Built for deterministic playback and strong movie workflow support.
- Technical details are in [GB/GBA Core](../../~docs/GB-GBA-CORE.md).
