# Sega Master System / Game Gear

## Overview

Nexen supports Sega Master System and Game Gear emulation with save states, rewind, movie recording, TAS editor support, and debugger tooling.

## Hardware Focus

- CPU: Z80-compatible behavior for SMS/GG workflows
- Video: VDP-driven display behavior
- Audio: PSG output and channel-level tools
- Cartridge: mapper and bank switching behavior

## Using SMS/GG Features in Nexen

- Open ROM from the File menu.
- Use save state workflows for quick testing.
- Use rewind during difficult sections or route validation.
- Use TAS editor and branches for frame-perfect planning.

## Accuracy and Compatibility

- Emphasis on stable gameplay and deterministic tooling.
- Low-level details are in [SMS/PCE/WS Core](../../~docs/SMS-PCE-WS-CORE.md).
