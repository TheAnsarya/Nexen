# PC Engine / TurboGrafx-16

## Overview

Nexen provides PC Engine and TurboGrafx-16 emulation with save states, rewind, movie support, TAS editing, and debugger tools.

## Hardware Focus

- CPU: HuC6280 behavior and addressing model
- Video: console rendering pipeline behavior
- Audio: system sound channel behavior
- Cartridge: ROM mapping and bank interactions

## Using PCE Features in Nexen

- Open ROM and verify system auto-detection.
- Use save states and rewind while testing difficult sequences.
- Record movies for deterministic replay and TAS editing.
- Use disassembler and memory viewer during debugging sessions.

## Accuracy and Compatibility

- Aims for stable and predictable behavior for gameplay and TAS.
- See [SMS/PCE/WS Core](../../~docs/SMS-PCE-WS-CORE.md) for internals.
