# Manual Testing Hub

This section is the starting point for manual release validation in Nexen.

## Start Here

1. Read the ranked checklist in [Manual Step Priority Ranking](manual-step-priority-ranking.md).
2. Execute all release-blocking steps first.
3. Run the platform workflow in [Atari 2600 Debug and TAS Manual Test](atari2600-debug-and-tas-manual-test.md).
4. Record pass and fail evidence in the active session log.

| [Atari 2600 Debug and TAS Manual Test](atari2600-debug-and-tas-manual-test.md) | End-to-end operator checklist for debugger UI, TAS UI, and movie workflows |

## Release Rule

Do not mark Atari 2600 release-ready unless every Critical step and every High step passes.
