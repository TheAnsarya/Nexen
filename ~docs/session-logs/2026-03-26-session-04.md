### 3. Tile Viewer Architecture Research

Deep-dived into `GetTileView()` across all platforms:

- `GetTileView()` is **non-virtual** — fully implemented in `PpuTools` base class
- It dispatches to `InternalGetTileView<format>()` → `GetTilePixelColor<format>()`
- Platforms don't override GetTileView; they declare supported `TileFormat` values
- Lynx was configured with `TileFormat::Bpp4` (SNES planar) — **WRONG**
- Lynx framebuffer uses 4bpp packed nibbles (high-nibble-first)
- `WsBpp4Packed` matches exactly: even pixel = high nibble, odd pixel = low nibble
