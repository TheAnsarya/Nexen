## Fixes Detail

1. **[HIGH] GetFrameInput shared mutable cache** — `_frameInputCache` was cleared and rebuilt per call; Lua scripts holding references to old results saw silently overwritten data
2. **[MEDIUM] Keyboard shortcut TextBox interception** — Space/F/R keys intercepted even when SearchBox had focus, preventing users from typing in search
3. **[MEDIUM] Dialog deadlock** — TasInputDialog and TasSelectRangeDialog left TaskCompletionSource incomplete if no parent window existed
4. **[MEDIUM] Range validation** — TasSelectRangeDialog accepted reversed ranges (From=100, To=50)
5. **[MEDIUM] Silent menu failures** — CreateNewTasMovie/StartTasRecording silently failed without feedback
6. **[MEDIUM] MovieRecord constructor crash** — EmuApi.GetRomInfo() called unconditionally in constructor
