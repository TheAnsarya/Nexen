## Implementation Details (#1904)

- Added `HasSegaHeader(...)` helper to guard SMD heuristics against false positives.
- Added `IsLikelySmdImage(...)` using:
  - explicit `.smd` extension detection
  - heuristic detection for copier-header + 16KB interleaved block layout
  - linear-header suppression when `SEGA` header already appears at `$0100`.
- Added `DecodeSmdToLinear(...)`:
  - full 16KB block deinterleave path
  - irregular payload fallback that strips 512-byte copier header only.
- Improved region-header parsing to consider full `$01F0-$01FF` marker span and hex-bitmask style region markers used by some dumps.
