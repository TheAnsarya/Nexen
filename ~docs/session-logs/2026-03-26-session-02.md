### 6. APU `_channelTickableMask` Optimization

Implemented enabled-channel fast-skip in `LynxApu`, mirroring Mikey's `_tickableMask` pattern:

- Added `uint8_t _channelTickableMask` tracking which channels are enabled (Control bit 3)
- `Tick()` uses mask to skip disabled channels with only `LastTick = currentCycle` (no function call)
- Mask updated on `Init()`, `WriteRegister()` (Control register), and `Deserialize()`
