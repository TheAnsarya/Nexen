## Implemented Changes

1. Added mapper state to `GenesisMemoryManager`:
	- `_romBankMapperEnabled`
	- `_romBankRegisters[7]`
2. Added mapper helper APIs:
	- `ResetRomBankMapper()`
	- `TryGetRomBankRegisterSlot()`
	- `TryWriteRomBankRegister()`
	- `TranslateRomAddress()`
3. Added ROM address translation for runtime ROM reads:
	- `Read8` now maps ROM addresses via `TranslateRomAddress`.
	- `Read16` now maps high/low bytes via `TranslateRomAddress`.
4. Added mapper register read support:
	- Runtime `Read8` can read active mapper bank register values (`A130F3..A130FF` odd addresses).
	- `Read16` handles mapper register region by delegating through `Read8`.
5. Added mapper register write support:
	- Runtime `Write8` now processes mapper bank register writes.
	- Runtime `Write16` now supports mapper register writes through byte-split writes.
6. Added debug path mapper behavior:
	- `DebugRead8` uses translated ROM addresses for ROM space.
	- `DebugRead8` can read mapper bank registers.
	- `DebugWrite8` updates mapper bank registers and records transcript entries.
7. Added mapper reset lifecycle integration:
	- `Init()` initializes default mapper bank layout.
	- `ResetRuntimeState()` resets mapper bank state to default mapping.
8. Added save-state support for mapper state:
	- `Serialize()` now saves/loads mapper enabled flag and bank register values.
9. Updated address introspection behavior:
	- `GetAbsoluteAddress()` now maps ROM CPU addresses through `TranslateRomAddress`.
10. Kept all changes semantics-focused on real runtime behavior, with no rename-only cleanup wave work.
