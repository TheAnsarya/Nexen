## Behavioral Changes

- VINT event now tracks explicit pending/new internal state:
	- Pending is latched at deferred VBlank event point even if VINT is masked.
	- New IRQ request is generated when enabled and pending (including enable-edge transitions).
- HINT event now tracks explicit pending/new internal state:
	- Pending latches on HINT counter roll-over.
	- New IRQ request is generated when enabled and pending (including enable-edge transitions).
- IRQ acknowledge now consumes the active VDP source similarly to expanded behavior:
	- Level 6 consume path for VINT pending/new.
	- Level 4 consume path for HINT pending/new.
- Status-read clear behavior remains aligned with existing Nexen model while now synchronizing internal VINT pending/new state.
