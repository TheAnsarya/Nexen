## Technical Changes

- Added latched core DMA transfer state in `GenesisVdp`:
	- `_dmaInitialized`
	- `_dmaLatchedMode`
	- `_dmaRemainingWords`
	- `_dmaSourceAddress`
	- `_dmaCopySourceAddress`
- Added `GetDmaWordsPerScanline()` with deterministic pacing budget:
	- Active display: 14 words/scanline
	- Blanking/display-off: H40=107 words/scanline, H32=85 words/scanline
- Updated DMA trigger path in `WriteControlPort` to reset latched DMA state when a new DMA starts.
- Reworked `ProcessDma()`:
	- Latches mode/length/source once at transfer start.
	- Applies pacing budget for 68K-bus DMA (`dmaMode 0/1`) and keeps `DmaBusy` asserted until completion.
	- Updates DMA source registers incrementally during paced 68K copy.
	- Preserves immediate completion for internal fill/copy (`dmaMode 2/3`) to maintain existing object-safety expectations.
	- Updates R19/R20 length registers while work remains, then clears on completion.
- Added serialization for new DMA latch fields.
