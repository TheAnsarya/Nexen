## Changes Made

### InternalRegisters.h — ProcessIrqCounters

- Added `[[likely]]` fast-path for `hClock > 10` with early return (~97% of calls)
- Added `[[unlikely]]` on `_needIrq > 0` branch

### SpcTimer.h — Run()

- Added `[[unlikely]]` on timer overflow branch (rare: ~1 in 16-128 calls)

### SnesMemoryManager.h/.cpp — Dead code removal

- Removed `Func` typedef, `_execRead`/`_execWrite` function pointer fields
- Removed `UpdateExecCallbacks()` declaration and implementation
- Simplified `SetCpuSpeed()` to just `_cpuSpeed = speed`
- Removed stale `UpdateExecCallbacks()` call from `Initialize()` and `Serialize()`

### SnesPpu.h/.cpp — ApplyColorMathToPixel template specialization

- Changed `ApplyColorMathToPixel` from runtime `bool subtractMode` parameter to `template <bool subtractMode>`
- Replaced `if (subtractMode)` with `if constexpr (subtractMode)` for compile-time branch elimination
- Hoisted subtract/add dispatch outside per-pixel loops in `ApplyColorMath()` (both normal and hi-res paths)

### SnesPpu.cpp — RenderSprites early exit

- Added `_spriteCount == 0` early exit with `[[unlikely]]` after `IsRenderRequired` check
