## Work Done

### Issue #1176 — Startup Crash Fix (commit `caa718e7`)

Users on Windows 11 24H2 LTSC and Linux reported that Nexen becomes a zombie process after the setup wizard completes. Root cause: `EmuApi.InitDll()` in the `MainWindow` constructor had no try/catch — any failure propagated unhandled, and the UI thread exception handler tried to show an Avalonia dialog which also silently failed.

**Changes:**

- **MainWindow.axaml.cs**: Wrapped `EmuApi.InitDll()` in try/catch with fatal logging before re-throwing
- **App.axaml.cs**: `InitializeMainWindow` retry path now catches second failure, closes splash, shows `ShowFatalStartupError` dialog with actionable troubleshooting info
- **App.axaml.cs**: Added `ShowFatalStartupError` with Win32 `MessageBoxW` native fallback when Avalonia rendering is broken
- **App.axaml.cs**: `UnhandledException` handler now logs before showing dialog and catches dialog failures
- **Program.cs**: Added OS version, .NET version, architecture, app base path to startup diagnostics in `nexen-log.txt`

Commented on #1176 requesting `nexen-log.txt` from the user. Also linked #1039 as likely same root cause.

### Yellow Pause Badge (commit `7f265326`)

Added a yellow ⏸ badge to save states that were saved while the emulator was paused. This required changes across the full stack:

- **SaveStateManager.h**: Added `bool isPaused` to `SaveStateInfo` struct
- **SaveStateManager.cpp**: `GetSaveStateList()` now reads the pause byte (last byte of v5+ save files) for each state
- **EmuApiWrapper.cpp**: Added `uint8_t isPaused` to `InteropSaveStateInfo`, populated in both `GetSaveStateList` and `GetRecentPlayStates`
- **EmuApi.cs**: Added `byte IsPaused` to C# `InteropSaveStateInfo`, `bool IsPaused` to `SaveStateInfo` class
- **RecentGamesViewModel.cs**: Added `bool IsPaused` to `RecentGameInfo`, populated from `SaveStateInfo`
- **StateGridEntry.axaml.cs**: Added `ShowPauseBadge` styled property, set from `game.IsPaused` in `Init()`
- **StateGridEntry.axaml**: Added second badge Border at top-right with yellow background and ⏸ text
