## Technical Changes

- Added latched startup-delay state for core DMA progression:
	- `_dmaStartupDelayLinesRemaining`
- 68K DMA trigger now asserts `DmaBusy` immediately when DMA is armed.
- Implemented differentiated startup delay for 68K-bus DMA mode:
	- H40 mode: 1 line delay
	- H32 mode: 2 line delay
- During startup delay window:
	- DMA remains active
	- Remaining-length registers do not decrement
	- Busy bit remains set
- On completion:
	- `DmaActive` cleared
	- `DmaBusy` cleared
	- Remaining-length registers cleared
- Added serialization support for startup-delay state.
