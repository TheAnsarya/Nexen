## Work Done

### Epic #1376 — SNES Performance Optimization Phase 3

Researched and implemented 3 PPU rendering optimizations targeting full-scanline hot paths:

1. **#1377 PrecomputeWindowMasks skip memset** — When `activeCount == 0`, skip `memset` for layers 0–4 (mask is never read due to short-circuit guard). Layer 5 (`ColorWindowIndex`) keeps memset because `ApplyColorMath` reads `_windowMask[ColorWindowIndex][x]` unconditionally. Eliminates ~1.25KB dead stores per scanline.

2. **#1378 RenderBgColor InternalCgramAddress hoist** — Hoisted `_state.InternalCgramAddress = 0` out of the per-pixel loop. Uses a `hasBgPixel` flag to set once after the loop. Eliminates ~256 redundant stores per scanline.

3. **#1379 Sprite priority scan early-exit** — Added early-exit to the 255-iteration sprite priority scan. Tracks found priority level count and breaks when all 4 levels are discovered. Typical exit after ~20–40 iterations instead of 255.

4. **#1380 HasActiveDmaChannel bitmask** — DEFERRED. DmaActive is set/cleared at 11 locations across SnesDmaController.cpp. Converting to a bitmask would require touching all 11 sites with high regression risk.
