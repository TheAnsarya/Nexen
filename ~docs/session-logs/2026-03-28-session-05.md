- **Added Genesis memory regions:** ROM (0x000000–0x3FFFFF), Z80 Address Space, IO Registers, VDP Registers, Work RAM (0xFF0000–0xFFFFFF)

- **Added WonderSwan branch detection:** V30MZ Jcc opcodes (0x70–0x7F, 0x0F prefixed)
- **Added WonderSwan write detection:** MOV patterns (0x88, 0x89, 0xA2, 0xA3)
