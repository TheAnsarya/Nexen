## Runtime Changes

- Added line-latched VDP runtime fields:
	- `_lineDisplayEnabled`
	- `_lineH40Mode`
	- `_lineScreenWidth`
- Latched these fields at scanline boundaries in `GenesisVdp::Run`.
- Updated scanline rendering path to consume latched state in `ProcessScanline` and `RenderScanline`.
- Updated DMA pacing/gating to consume latched state in:
	- `GetDmaWordPeriodCycles`
	- `IsActiveDisplayExternalDmaSlot`
	- `ProcessDma`
- Deferred register-driven screen-width application to scanline boundaries.
- Added boundary handling in `WriteControlPort` so R1/R12 writes at exact line boundary immediately affect the upcoming line latch.
- Serialized new latch fields in `Serialize` for save-state determinism.
