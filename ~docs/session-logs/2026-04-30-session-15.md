## Technical Changes

- Replaced line-granularity startup delay with cycle-level startup delay fields:
	- `_dmaStartupDelayCyclesRemaining`
	- `_dmaBusCycleRemainder`
- Converted 68K bus DMA startup delay to cycle-based values with H40/H32 differentiation.
- Moved DMA progression to run per core cycle in `Run()` (instead of only per scanline boundary), enabling cycle-level transition behavior.
- Added cycle-period pacing for 68K bus DMA transfer words:
	- Faster cadence during blanking/display-off
	- Slower cadence during active display
- Preserved existing behavior for internal DMA modes (fill/copy) and deterministic completion semantics.
- Updated serialization for the new cycle-level DMA state.

## Test Updates
