# Session 16 — Remove ReactiveUI.Avalonia Bridge (#1213)

**Date:** 2026-04-12
**Branch:** master
**Focus:** Eliminate Avalonia 11/12 version skew from ReactiveUI.Avalonia bridge package

## Summary

Resolved high-priority issue #1213 by removing the `ReactiveUI.Avalonia 11.4.12` bridge package, which targeted Avalonia 11.x and caused major-version skew with Nexen's Avalonia 12.0.0.

## Analysis

### Bridge Usage Audit

Exhaustive search confirmed the bridge was **essentially unused**:

- **Zero** `ReactiveWindow<T>` or `ReactiveUserControl<T>` base classes
- **Zero** `RoutedViewHost` / `ViewModelViewHost` in XAML
- **Zero** `WhenActivated` on views
- **Zero** `BindCommand` / `OneWayBind` code-behind bindings
- **Zero** `IViewFor<T>` implementations
- **Zero** XAML namespaces referencing `ReactiveUI.Avalonia`
- **Single** usage: `.UseReactiveUI(_ => { })` in `Program.cs:336`

### ReactiveUI 23.x API Changes

Discovered that ReactiveUI 23.2.1 replaced the `RxApp` class:

- `RxApp.MainThreadScheduler` → `RxSchedulers.MainThreadScheduler`
- New builder API: `RxAppBuilder.CreateReactiveUIBuilder().WithMainThreadScheduler(...).BuildApp()`

### NuGet Version Landscape

- `ReactiveUI.Avalonia` latest stable: 11.4.12 (no Avalonia 12 version)
- `Avalonia.ReactiveUI` latest stable: 11.3.8 (no Avalonia 12 version)
- `Avalonia.ReactiveUI` nightly: 12.0.999-cibuild0060150-alpha (unstable CI build only)
- `Avalonia.BuildServices 11.3.2` is Avalonia 12's own transitive dependency (not from bridge)

## Changes

### New Files

- `UI/Utilities/AvaloniaDispatcherScheduler.cs` — Minimal `LocalScheduler` implementation wrapping `Dispatcher.UIThread` for immediate scheduling and `DispatcherTimer.RunOnce` for deferred scheduling

### Modified Files

- `UI/UI.csproj` — Removed `ReactiveUI.Avalonia 11.4.12` package reference
- `UI/Program.cs` — Replaced `using ReactiveUI.Avalonia` + `.UseReactiveUI(_ => { })` with `using ReactiveUI.Builder` + `.AfterSetup(_ => RxAppBuilder.CreateReactiveUIBuilder().WithMainThreadScheduler(...).BuildApp())`
