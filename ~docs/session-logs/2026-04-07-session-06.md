## Changes

### Bug Fix — Games Starting Paused (#1184)

**Root cause:** `DebuggerWindow.axaml.cs` GameLoaded handler unconditionally paused emulation and only auto-resumed if `BreakOnPowerCycleReset` was false (default: true). This conflated fresh game loads with power cycle resets.

**Fix:** Added `evtParams.IsPowerCycle` check so fresh game loads always auto-resume. Power cycle behavior still respects `BreakOnPowerCycleReset`.

**File:** `UI/Debugger/Windows/DebuggerWindow.axaml.cs`
