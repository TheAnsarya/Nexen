## Final Code Change

In `SnesCpu::Compare(uint16_t reg, bool eightBitMode)`, carry is now derived from subtraction result:

- 8-bit path: `flags = (uint8_t)(result <= (uint8_t)reg)`
- 16-bit path: `flags = (uint8_t)(result <= reg)`

This replaces explicit operand comparisons (`reg >= value`) while preserving semantics.
