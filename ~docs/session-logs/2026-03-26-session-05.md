### 3. APU MixOutput Fast-Skip (#959 ✅)

Added early-return fast-skip to `LynxApu::MixOutput()`:

- When `_state.Stereo == 0xff` (all channels muted on both sides), writes zero samples directly and returns
- Uses `[[unlikely]]` attribute since most games have audio enabled
- Skips the full 4-channel mix loop with 16+ STEREO/PAN branch tests per sample
- **42% faster** on muted path (2.96ns vs 7.01ns median), normal path completely unaffected

Added 4 new stereo muting tests:

- `Stereo_AllMuted_StereoRegister0xFF`
- `Stereo_PartialMute_LeftOnly` (0xF0)
- `Stereo_PartialMute_RightOnly` (0x0F)
- `Stereo_NotFullyMuted_HasAudio` (0xFE)

Added `BM_LynxApu_StereoMix_AllMuted` benchmark.
