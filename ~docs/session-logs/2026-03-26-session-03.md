### 3. Performance: Cart Address Bitmask ‚Äî Issue #958

Replaced modulo division with bitwise AND for cart address wrapping in `LynxCart`:

- Added `_bank0Mask` / `_bank1Mask` fields to `LynxCart.h`
- Precompute masks in `Init()`: `mask = (size > 0) ? (size - 1) : 0`
- Masks recomputed after bank size adjustment block (edge case fix)
- `GetCurrentRomAddress()`: `addr % bankSize` ‚Üí `addr & bankMask`
- Benchmark: ~4.86ns ‚Üí ~3.82ns (~21% faster)

### 4. Performance: Suzy Pixel Write Unchecked ‚Äî Issue #958

Removed redundant bounds check from `WriteSpritePixel()`:

- Caller (`ProcessSprite`) already checks both X (line 417) and Y (line 393) bounds
- `LeftHand` mirror preserves x in [0, ScreenWidth)
- Removed the `unsigned(x) >= ScreenWidth || unsigned(y) >= ScreenHeight` check
- Benchmark: ~5.96ns ‚Üí ~4.68ns (~21% faster)
