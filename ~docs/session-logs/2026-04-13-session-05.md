## Changes

### Issue #1251 ‚Äî Linux SIGSEGV Crash Fix

**Root cause:** `_controllers` vector in `LinuxKeyManager` accessed concurrently by the emulation thread (`SetForceFeedback`, `IsKeyPressed`, `GetAxisPosition`, `GetPressedKeys`) and `_updateDeviceThread` (`erase`/`push_back` every 5 seconds) with zero synchronization.

**Fix:**

- Added `std::mutex _controllerMutex` to `LinuxKeyManager.h`
- Added `std::lock_guard<std::mutex>` in all 6 methods accessing `_controllers`
- Added bounds check in `GetAxisPosition` for out-of-range port
- Removed dead `controllersToAdd` variable

**Files:** `Linux/LinuxKeyManager.h`, `Linux/LinuxKeyManager.cpp`
