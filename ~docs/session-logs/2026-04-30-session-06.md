	- Helps correlate spontaneous state collapse with explicit console reset calls.

- `GenesisDebugger::SetProgramCounter()`:
	- Logs debugger-driven PC mutations when they affect CPU state (`updateDebuggerOnly=false`).
	- Includes old/new PC and identifies when PC is explicitly set to zero.
