## Technical Changes

- In ReadDataPort and WriteDataPort, switched mode dispatch from full command code to low nibble via `mode = _accessMode & 0x0F`.
- In WriteControlPort second-word decode, updated command-code assembly to:
  - `_accessMode = (full >> 14) | ((value >> 2) & 0x3C)`
- Updated DMA trigger condition to use command code bit 5 (`_accessMode & 0x20`) instead of raw second-word bit check.
