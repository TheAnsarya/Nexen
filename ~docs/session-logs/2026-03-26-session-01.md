#### LynxDisUtils Performance Optimization ‚Äî #949

Replaced switch statements and double indirection with constexpr lookup tables:

1. **`_opSizeDirect[256]`** ‚Äî Pre-computed opcode‚Üísize table eliminates `_opMode[opCode]` ‚Üí `_opSize[mode]` double indirection
2. **`_opClassify[256]`** ‚Äî Combined flags table (UNCONDITIONAL, CONDITIONAL, JUMP_TO_SUB, RETURN) replaces 4 separate switch statements
3. **`GetEffectiveAddress`** ‚Äî Single table lookup + bitmask instead of two function calls
4. **`GetOpFlags`** ‚Äî Uses both tables directly, eliminating 3 function calls
