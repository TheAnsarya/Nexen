### Performance Improvements (#1170)

1. **FullscreenResolution lookup optimization** (`UI/Config/VideoConfig.cs`)
	- Replaced string parsing (`ToString()` + `Substring()` + `Int32.Parse()`) with static `Dictionary<FullscreenResolution, (int, int)>` lookup
	- Eliminates all string allocations on every `GetWidth()`/`GetHeight()` call

2. **WatchManager.FormatValue hex fix** (`UI/Debugger/WatchManager.cs`)
	- Fixed hex format specifier from uppercase `"X"` to lowercase `"x"` per coding standards
	- Used string interpolation instead of string concatenation

3. **WatchManager.RemoveWatch optimization** (`UI/Debugger/WatchManager.cs`)
	- Replaced LINQ `.Where().ToList()` with in-place backward `RemoveAt()` loop
	- Eliminates intermediate list allocation
